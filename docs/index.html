StudioYoga Manager (Projeto em C++)

Bem-vindo à página do projeto StudioYoga Manager. Este é um sistema de console (desktop) desenvolvido em C++ para auxiliar no gerenciamento de um estúdio de yoga.

O que o projeto faz?

O programa permite o cadastramento, consulta e gerenciamento de:

  • Alunos
  • Professores
  • Aulas e horários

Como usar?

Este projeto não é um site, mas sim um programa em C++. Para usá-lo, você precisa baixar o código-fonte, compilá-lo e executá-lo no seu computador (via terminal).

1. Clone o repositório:

git clone https://github.com/laissaraiva/StudioYoga-Manager-cpp.git

2. Navegue até a pasta e compile o código (exemplo com g++):

cd StudioYoga-Manager-cpp
g++ -o yoga_manager *.cpp

3. Execute o programa:

./yoga_manager
