O Studio Yoga Manager é uma solução de software robusta, desenvolvida em C++, projetada para centralizar e simplificar a administração completa de um estúdio de yoga. O sistema oferece uma interface de console (CLI) para gerenciar de forma intuitiva todas as entidades e operações do dia a dia, desde o cadastro de alunos até o agendamento de aulas.

O pilar do desenvolvimento foi a Programação Orientada a Objetos (POO). Conceitos avançados foram aplicados para garantir um código limpo, modular e extensível:

  • Herança: Utilizada para criar hierarquias de classes lógicas, como a classe base Pessoa que define atributos comuns para Praticante e Instrutor.
  • Polimorfismo: Implementado de forma crucial no sistema de aulas, onde a classe base abstrata Aula permite ao sistema gerenciar diferentes tipos de aulas (como HotYoga e YogaPets) de maneira uniforme.

Adotando práticas de Clean Architecture, o projeto foi rigorosamente modularizado em camadas de responsabilidade distintas, refletidas na estrutura de pastas:

  • models/: Contém as definições das entidades centrais (as classes Plano, Aula, Praticante, etc.).
  • services/: O "cérebro" da aplicação. O StudioManager reside aqui, controlando toda a lógica de negócio, o fluxo do menu e gerenciando os dados em memória.
  • data/: A camada de persistência. O DataManager é um serviço estático responsável por salvar (serializar) e carregar (desserializar) os dados dos vetores em arquivos .txt, garantindo que as informações sobrevivam entre as execuções.
  • validator/: Uma camada de serviço utilitário que garante a integridade dos dados de entrada (como nomes, emails e preços) antes que eles sejam processados pela camada de serviço.

O resultado é um sistema de gerenciamento funcional, que não apenas implementa um CRUD (Create, Read, Update, Delete) completo para as entidades do estúdio, mas também garante a persistência de dados e a validação de entradas. O projeto utiliza CMake para um processo de compilação (build) moderno e multiplataforma, estando pronto para ser compilado e executado.

Applications (Como usar o programa)

Este projeto não é um site, mas sim um programa em C++ que roda no seu terminal. Para usá-lo, você precisa baixar o código-fonte e compilá-lo usando CMake (o comando g++ simples não funcionará devido à estrutura de pastas).

1. Clone o repositório:

git clone https://github.com/laissaraiva/StudioYoga-Manager-cpp.git

2. Entre na pasta e prepare a compilação (Build):

cd StudioYoga-Manager-cpp
mkdir build
cd build
cmake ..

3. Compile o projeto:

cmake --build .

4. Execute o programa:

./StudioYogaManager
