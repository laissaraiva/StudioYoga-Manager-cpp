Abstract

Este projeto apresenta o desenvolvimento de um sistema de informação para gerenciamento de um estúdio de yoga, utilizando a linguagem C++ e aplicando os princípios da Programação Orientada a Objetos (POO). O sistema permite o cadastro, consulta e gerenciamento de alunos, professores e aulas, visando otimizar a organização e o fluxo de trabalho do estúdio.

Applications (Como usar o programa)

Este projeto não é um site, mas sim um programa em C++. Para usá-lo, você precisa baixar o código-fonte...

git clone https://github.com/laissaraiva/StudioYoga-Manager-cpp.git
cd StudioYoga-Manager-cpp
g++ -o yoga_manager *.cpp
./yoga_manager
