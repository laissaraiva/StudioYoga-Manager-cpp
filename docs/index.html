StudioYoga Manager (Projeto em C++)

Bem-vindo à página do projeto. Este é um sistema de console (desktop) desenvolvido em C++.

O que o projeto faz?

O programa permite o cadastramento, consulta e gerenciamento de:

  • Alunos
  • Professores
  • Aulas e horários

Como usar?

Este projeto não é um site, mas sim um programa em C++. Para usá-lo, você precisa baixar o código-fonte...

git clone https://github.com/laissaraiva/StudioYoga-Manager-cpp.git
cd StudioYoga-Manager-cpp
g++ -o yoga_manager *.cpp
./yoga_manager
